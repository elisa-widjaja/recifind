# Saving recipes to ReciFriend — quick guide

## Supported sources

| | |
|---|---|
| ✅ TikTok | reels, videos |
| ✅ Instagram | reels, posts |
| ✅ YouTube | videos and shorts |
| ✅ Pinterest | pins |
| ✅ Recipe sites | AllRecipes, NYT Cooking, Fresh off the Grid |
| ✅ Google Docs | any link-shared doc |
| ❌ Facebook | not yet supported |

## Two ways to save (iPhone, app installed)

### 1. Share Extension (fastest)

- Open the reel/post in the TikTok or Instagram app
- Tap **Share** → scroll the app row and tap **ReciFriend**
- ReciFriend opens, parses, and saves it

### 2. Paste the link

- In TikTok/Instagram, tap **Share** → **Copy link**
- Open ReciFriend → tap the **+** Add Recipe button
- Paste the link → tap **Save**

## On web (recifriend.com) or Android

Only the paste-the-link flow is available. Same steps as Method 2.

## Tips for best results

- Reels where the creator lists ingredients in the **caption** parse the cleanest
- Save from the **original creator's post**, not a repost — reposts often strip captions
- If something imports with missing pieces (no ingredients, no steps), you can edit any field manually
